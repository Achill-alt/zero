## 1. 阶段锚定

本阶段执行工程初始化和各模块骨架搭建：

`工程初始化 → 数据库建表 → JWT中间件 → 全部API骨架 → Vue项目搭建 → 全部页面骨架`

范围内：
- Python + FastAPI 工程骨架（main.py, config.py, database.py）
- SQLAlchemy 全部模型（User, Contract, ContractTemplate, ApprovalChainTemplate, ApprovalInstance, AuditLog）
- FTS5 全文索引初始化和触发器
- Pydantic 请求/响应 Schemas
- JWT + bcrypt 认证中间件和角色权限中间件
- 全部 API 路由（auth, contracts, approvals, approval_chains, search, templates, users, audit_logs）
- 业务 Service 层（auth_service, contract_service, approval_service, search_service, audit_service）
- Seed 脚本（管理员 + 4个审批人 + 2个审批链模板）
- pytest 测试框架（3 tests passed）
- Vue 3 + Vite + Element Plus + Pinia + Vue Router 前端项目
- 12 个页面组件（Login, Dashboard, ContractList, ContractCreate, ContractDetail, ContractEdit, ApprovalCenter, TemplateManage, ExpiringPanel, UserManage, ApprovalChainConfig, AuditLogs）
- 3 个布局组件（AppLayout, AppNavbar, AppSidebar）
- Axios API 封装（auth, contracts, approvals, admin）
- Vite 代理配置（/api → localhost:8000）

范围外：
- 无（M0 已完成全部计划内容）
