| **UJ-9** 权限校验 | handler 访问 admin API | 403 拒绝 |

### 3.2 状态机验证

```
draft → pending_approval (提交审批)
draft → voided (作废)
pending_approval → draft (驳回)
pending_approval → draft (撤回)
pending_approval → approved (全部审批通过)
approved → archived (归档)
voided → 不可提交 (409)
```
