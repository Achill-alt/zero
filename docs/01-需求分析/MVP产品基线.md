# 企业合同管理系统 MVP 产品基线

- 文档版本：v1.0
- 文档状态：已确认

- 适用范围：企业合同全生命周期管理 Web 应用 MVP
- 编制依据：本项目头脑风暴过程中用户逐项确认的结论

> 本文是经用户确认的 MVP 产品基线，可作为后续 PRD、SRS 和页面交互需求编制依据。进入系统设计或编码前，仍需完成并确认相应需求文档。

## 1. 产品定位

面向企业内部的合同全生命周期管理系统。覆盖合同拟制 → 多级审批 → 归档 → 到期预警 → 全文检索的完整链路，核心亮点为全动态可配置审批链引擎。

## 2. MVP 目标

2 天 5 人 vibecoding 交付一个可演示、可答辩的企业合同管理系统。全链路跑通，每个功能做最简实现，审批链引擎作为答辩核心亮点。

## 3. 目标用户与角色

| 角色 | 说明 |
|------|------|
| 经办人 | 拟制合同、提交审批、查看审批进度、合同归档 |
| 审批人 | 查看待审批合同、通过/驳回（附理由）、查看审批历史 |
| 管理员 | 用户管理、角色分配、审批链模板配置、审计日志查看 |

MVP 通过 JWT 认证实现真实角色切换。

## 4. 业务范围

首版覆盖以下合同管理场景：

- 合同拟制（富文本内容 + 结构化字段）
- 合同模板管理（预设模板，拟制时可选择）
- 动态多级审批（可配置审批链，条件匹配）
- 审批流转（逐级通过/驳回/撤回）
- 合同归档
- 到期预警（前端预警面板）
- 全文检索（SQLite FTS5）
- 审计日志（关键操作全记录）

## 5. 核心流程

1. 经办人登录系统，选择合同模板或空白合同拟制。
2. 填写合同标题、内容、金额、类型、起止日期等字段后提交审批。
3. 系统根据合同条件（金额、类型等）自动匹配审批链模板，生成审批实例。
4. 审批链按配置的步骤逐级流转，每级审批人通过/驳回（附理由）。
5. 驳回时合同退回经办人修改后重新提交；经办人可随时撤回审批中的合同。
6. 全部审批通过后，合同进入待归档状态，经办人完成归档。
7. 系统根据合同截止日期自动计算到期状态，预警面板展示即将到期合同。
8. 全文检索支持按关键词、合同类型、日期范围搜索已归档和审批中的合同。

## 6. 合同字段

### 6.1 合同基础字段

- 合同标题
- 合同内容（富文本）
- 合同类型（采购/销售/服务/租赁/其他）
- 合同金额
- 甲方/乙方
- 开始日期
- 结束日期
- 经办人
- 当前状态（草稿/审批中/已归档/已作废）

### 6.2 合同类型固定选项

- 采购合同
- 销售合同
- 服务合同
- 租赁合同
- 其他

## 7. 审批链引擎（核心亮点）

### 7.1 审批链模板配置

- 管理员可创建、编辑、启用/禁用审批链模板。
- 每个模板包含：名称、触发条件（JSON）、审批步骤序列（JSON）。
- 触发条件支持：合同金额范围、合同类型、经办人部门等。
- 审批步骤包含：步骤名称、审批角色、超时时间（小时）。
- 多个模板时按优先级匹配。

### 7.2 审批流转

- 合同提交时，系统按优先级匹配审批链模板，生成审批实例。
- 审批按步骤顺序逐级流转，当前步骤审批人通过后自动进入下一步。
- 任意步骤驳回时，合同回到经办人草稿箱，填写驳回理由。
- 经办人可修改后重新提交或撤回审批。
- 全部步骤通过后，合同进入待归档状态。

### 7.3 审批角色

审批步骤中指定角色而非具体用户，审批时查询该角色当前成员：

- dept_manager：部门负责人
- legal：法务
- finance_director：财务总监
- ceo：总经理

管理员可为用户分配以上角色。

## 8. 全文检索

- 基于 SQLite FTS5 实现中文全文检索。
- 支持按关键词搜索合同标题和内容。
- 支持按合同类型、日期范围、状态筛选。
- 搜索结果高亮匹配关键词。

## 9. 到期预警

- 系统根据合同结束日期自动计算到期状态。
- 前端预警面板展示：30天内到期、已过期合同列表。
- 按紧急程度（已过期 > 7天内 > 30天内）排序。
- 预警面板作为独立页面，提供跳转到合同详情的能力。

## 10. 审计日志

- 记录所有关键操作：登录、合同创建/编辑/提交/审批/驳回/归档。
- 日志字段：操作用户、操作类型、目标类型、目标ID、操作详情（JSON）、操作时间。
- 管理员可在审计日志页面查看和筛选。

## 13. 运行边界

MVP 明确包含：
- JWT 登录/注册、角色管理
- 合同全生命周期管理
- 动态审批链引擎
- 全文检索
- 到期预警面板
- 审计日志
- 合同模板管理
- SQLite 数据持久化

MVP 明确不包含：
- 邮件/短信通知
- 电子签章
- 文件附件上传
- 移动端适配
- 对接外部系统（OA、ERP 等）
- 数据导入导出
- 合同版本历史
- 国际化
