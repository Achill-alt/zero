## 1. 项目启动

### 1.1 选题确定

**用户提出**

> 企业合同管理系统（推荐），合同全生命周期管理：拟制 → 多级审批 → 归档 → 到期预警 → 全文检索。

核心维度：
- 核心流程：合同拟制 → 逐级审批（可配置层级）→ 归档 → 到期自动预警
- 角色：经办人、审批人、管理员
- 5人分工：认证+用户体系、合同CRUD+模板、审批链引擎、全文检索+归档、前端页面+预警面板
- 技术亮点：可配置审批链、关键词全文检索、到期预警机制、审计日志

### 1.2 项目背景澄清

## 4. 设计确认

### 4.1 总体架构

前后端分离架构：Vue 3 (Vite) + FastAPI + SQLite (FTS5) + SQLAlchemy + JWT。

5人分工：认证+用户体系、合同CRUD+模板、审批链引擎、全文检索+归档、前端页面+预警面板。

### 4.2 审批链引擎

全动态配置：审批链模板支持多条件触发（金额、类型等），审批步骤的有序列表（角色+超时），状态机支持通过/驳回/撤回。

### 4.3 数据模型

核心表：users、contracts、approval_chain_templates、approval_instances、audit_log。

### 4.4 开发计划

2天：Day1 上午工程初始化+模块骨架，下午核心链路联调；Day2 上午完善功能+前端页面，下午收尾+答辩准备。
