# 企业合同管理系统 页面与交互说明

- 文档版本：v1.0
- 文档状态：已确认
- 编制日期：2026-07-08
- 适用版本：企业合同管理系统 MVP
- 编制依据：`docs/01-需求分析/MVP产品基线.md` v1.0 已确认

> 本文定义 MVP 所有页面的布局、组件、交互流程和路由设计。

## 1. 页面清单与路由

| 路由 | 页面 | 权限 | 说明 |
|------|------|------|------|
| `/login` | 登录页 | 公开 | 用户名密码登录 |
| `/register` | 注册页 | admin | 管理员注册新用户 |
| `/dashboard` | 工作台 | 认证用户 | 合同概览和快捷入口 |
| `/contracts` | 合同列表 | 认证用户 | 搜索、筛选、分页 |
| `/contracts/create` | 合同拟制 | handler | 新建合同 |
| `/contracts/:id` | 合同详情 | 认证用户 | 合同内容和审批历史 |
| `/contracts/:id/edit` | 合同编辑 | handler | 编辑合同（草稿/退回） |
| `/approvals` | 审批中心 | approver | 待审批合同列表 |
| `/templates` | 模板管理 | 认证用户 | 合同模板列表和创建 |
| `/expiring` | 预警面板 | 认证用户 | 到期/已过期合同 |
| `/admin/users` | 用户管理 | admin | 用户列表和角色管理 |
| `/admin/approval-chains` | 审批链配置 | admin | 审批链模板管理 |
| `/admin/audit-logs` | 审计日志 | admin | 操作日志查看 |

## 3. 页面详细说明

### 3.1 登录页 `/login`

- 居中卡片布局，Logo + 系统名称
- 用户名输入框、密码输入框
- 登录按钮
- 登录失败时红色提示
- 登录成功后跳转到工作台

### 3.2 工作台 `/dashboard`

- 顶部统计卡片行：我的合同总数、审批中数量、待我审批数量、即将到期数量
- "快捷操作"按钮组：拟制合同、审批中心、预警面板
- 最近合同列表（最近5条，显示标题、状态、创建时间）
- 即将到期合同提醒（3条）

### 3.3 合同列表 `/contracts`

- 顶部工具栏：状态筛选（下拉）、类型筛选（下拉）、搜索输入框、搜索按钮
- 表格列：合同标题、类型、金额、状态、经办人、创建时间
- 每行操作：查看详情、编辑（草稿/退回状态）
- 底部分页器
- 点击行或"查看详情"跳转到合同详情

### 3.4 合同拟制 `/contracts/create`

- 表单布局：
  - 模板选择（下拉，可选，选择后自动填充标题和内容）
  - 合同标题（文本输入）
  - 合同类型（下拉选择：采购/销售/服务/租赁/其他）
  - 甲方/乙方（文本输入）
  - 合同金额（数字输入）
  - 开始日期 / 结束日期（日期选择器）
  - 合同内容（富文本编辑器，基于 Quill 或 Tiptap）
- 底部按钮：保存草稿、提交审批
- 保存草稿成功提示，留在当前页
- 提交审批成功跳转到合同详情

### 3.5 合同详情 `/contracts/:id`

- 内容区展示全部合同字段
- 审批历史时间线（Step 组件）：
  - 每步显示：步骤名、审批人、结果（通过/驳回/待审批）、时间、评论
  - 当前步骤高亮
- 底部操作区（按状态）：
  - 草稿：编辑、提交审批、作废
  - 审批中（经办人）：撤回
  - 审批中（当前审批人）：通过、驳回（弹出理由输入框）
  - 已通过（经办人）：归档
  - 已归档：只读

### 3.6 合同编辑 `/contracts/:id/edit`

- 与拟制页相同表单，预填现有数据
- 保存后返回详情页

### 3.7 审批中心 `/approvals`

- 待审批合同表格：
  - 列：合同标题、类型、金额、经办人、提交时间、当前步骤
  - 每行：通过按钮（绿色）、驳回按钮（红色）
- 驳回弹出对话框：输入驳回理由（必填），确认/取消
- 通过直接确认，成功后该行消失
- 空状态："暂无待审批合同"

### 3.8 模板管理 `/templates`

- 模板列表卡片网格
- 每个卡片：模板名称、适用类型、创建时间
- 新建模板按钮 → 弹出对话框或跳转创建页
- 模板创建：名称、标题模板、内容模板（富文本）、适用类型

### 3.9 预警面板 `/expiring`

- 统计行：已过期数量、7天内到期数量、30天内到期数量
- 合同列表，按紧急程度分组：
  - 🔴 已过期
  - 🟠 7天内到期
  - 🟡 30天内到期
- 每项：合同标题、结束日期、距今天数标签、状态
- 点击跳转合同详情

### 3.10 用户管理 `/admin/users`

- 用户表格：用户名、显示名、角色、部门、状态、创建时间
- 新增用户按钮 → 注册弹窗
- 每行操作：编辑角色、启用/禁用

### 3.11 审批链配置 `/admin/approval-chains`

- 审批链模板卡片列表
- 每个卡片展示：名称、优先级、条件摘要、步骤数、启用状态
- 新建/编辑审批链 → 表单页面：
  - 模板名称
  - 优先级
  - 触发条件：合同类型（多选）、金额范围（min/max）
  - 审批步骤（动态列表，可增删）：
    - 每步：步骤名称、审批角色（下拉）、超时时间（小时）
    - 拖拽排序
  - 启用开关

### 3.12 审计日志 `/admin/audit-logs`

- 筛选栏：用户、操作类型、时间范围
- 日志表格：时间、用户、操作、目标、详情
- 底部分页器
- 只读，无编辑和删除操作

## 4. 交互通用规则

- 所有操作成功显示绿色 Message 提示
- 所有操作失败显示红色 Message 提示（含具体错误信息）
- 危险操作（作废、驳回）需要二次确认
- 表单提交时按钮显示 loading 状态，防止重复提交
- 列表为空时显示友好的空状态插图和文字
- 401 错误自动跳转登录页
- 顶部导航实时显示当前用户名
