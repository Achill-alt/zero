## 项目信息

### 项目名称

**企业合同管理系统** (Enterprise Contract Management System)

### 项目定位

毕设项目 · v1.2 · 全栈 Web 应用

覆盖合同全生命周期的数字化管理平台，支持合同拟制（富文本编辑器）、多级审批流转、全文检索、模板管理、到期预警、审计追溯、数据导出。

### 核心功能（14 项）

1. **认证与权限** — JWT 登录/注册，admin / handler / approver 三级角色，审批子角色（部门负责人→法务→财务总监→总经理）
2. **合同管理** — 富文本编辑器（TipTap）+ 从模板/自由创建，完整 CRUD，5 状态机（草稿→审批中→通过→归档/作废）
3. **附件上传** — 支持 PDF/Word/Excel/图片/TXT 上传、下载、删除
4. **审批引擎** — 优先级匹配 + JSON 条件驱动 + 多级顺序流转（通过/驳回/撤回），4 条审批链按金额/类型自动匹配
5. **全文检索** — jieba 中文分词 + SQLite FTS5 搜索（带高亮片段）
6. **模板管理** — 合同模板 CRUD，支持富文本内容模板（5 种类型）
7. **到期预警** — 30 天内即将到期合同自动识别
8. **审计日志** — 全操作记录 + 时间线可视化 + 中文映射 + 彩色图标
9. **用户管理** — 管理员可新增、启/禁用用户
10. **站内通知** — 审批事件触发通知（铃铛角标 + 通知列表）
11. **工作台** — 统计卡片 + 到期预警一览
12. **数据导出** — 合同列表导出 Excel (.xlsx) + PDF (.pdf) A4 卡片式布局
13. **审批链配置** — 可视化配置审批链的条件、步骤、优先级
14. **Precision Enterprise 设计** — 62 个 CSS 设计 Token + 企业级 UI/UX

### 合同状态机

```
                  ┌──────────┐
                  │   草稿   │ ← 编辑器可修改
                  └────┬─────┘
            提交审批 │     │ 作废
                    ↓     ↓
        ┌──────────────┐ ┌──────────┐
        │   审批中      │ │  已作废  │ → 不可操作
        └──────┬───────┘ └──────────┘
   通过 │     │ 驳回/撤回
        ↓     ↓ (退回草稿)
  ┌──────────────┐
  │   已通过     │
  └──────┬───────┘
   归档  │
        ↓
  ┌──────────────┐
  │   已归档     │ → 只读
  └──────────────┘
```

```
│   → 条件: contract_type=purchase                        │
│   → 步骤: dept_manager → ceo                            │
├─────────────────────────────────────────────────────────┤
│ Priority 10: 标准四级审批（兜底）                        │
│   → 条件: 全部类型                                      │
│   → 步骤: dept_manager → legal → finance_director → ceo │
├─────────────────────────────────────────────────────────┤
│ Priority 5: 小额合同简易审批                             │
│   → 条件: amount≤50K                                    │
│   → 步骤: dept_manager → finance_director               │
└─────────────────────────────────────────────────────────┘

匹配逻辑：遍历所有激活链 → 按条件打分 → 选最高优先级匹配
```
